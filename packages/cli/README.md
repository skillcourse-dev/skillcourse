# @skillcourse-dev/cli

Authoring CLI for [skillcourse](https://skillcourse.dev) courses.

```bash
npx @skillcourse-dev/cli init my-course
npx @skillcourse-dev/cli validate
npx @skillcourse-dev/cli bump patch
npx @skillcourse-dev/cli quiz init
```

See the [monorepo README](https://github.com/skillcourse-dev/skillcourse) for the full development guide.
